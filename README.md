# Wordpress Image Optimizer

A Wordpress plugin that optimizes images on the fly.

Build with [Antigravity](https://antigravity.google) and [BMad Method](https://github.com/bmad-method/bmad-method)

[![Tests](https://github.com/elnino1/wp-image-optimizer/actions/workflows/tests.yml/badge.svg)](https://github.com/elnino1/wp-image-optimizer/actions/workflows/tests.yml)

See the [HOW-TO](HOW-TO.md) for more information on how to use this plugin.

## Minimum requirements

- PHP 8.2
- Wordpress 6.4

## Repository structure

- `.agent/workflows/` — Antigravity slash-command workflows (`/pm`, `/dev`, `/qa`, ...)
- `AGENTS.md` — Codex-friendly agent memory (generated by BMAD-METHOD)
- `.gemini/transpose_bmad.py` — regenerates workflows from `.bmad-core`
